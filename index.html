Record ALL Browser Tabs?

For full browser monitoring (all tabs, all websites), install the extension:

  1. Download the extension folder from this project
  2. Open Chrome → chrome://extensions/
  3. Turn on "Developer mode" (top right)
  4. Click "Load unpacked" → Select the folder

This one-time setup takes 30 seconds. After that, it works automatically forever.
