Install Browser Extension

Required for full browser monitoring

The web app monitors this page only. Install the extension to capture keystrokes from ALL tabs automatically.

  1. 1 Open Chrome and type: chrome://extensions/
  2. 2 Turn ON "Developer mode" (top right switch)
  3. 3 Click "Load unpacked" button
  4. 4 Select the extension folder from this project
  5. 5 Click the extension icon and press "Start Monitoring"
After installation, all browser tabs will be monitored automatically.
